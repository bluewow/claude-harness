```
{{/}}                      [공개] 공개 셸
├─ {{/login}}              [공개]
└─ ({{app}})/              [로그인] 공통 셸: {{상단바·하단탭 등}}

   ├─ {{/...}}             S{{n}}
   └─ ({{admin}})/         [역할:{{관리자}}] {{미들웨어 게이트}}
      └─ {{/admin/...}}    S{{n}}
```
