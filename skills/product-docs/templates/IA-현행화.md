## 2. 라우트 그룹 · 레이아웃 셸 구조

> 공유 레이아웃(셸)과 라우트 그룹을 트리로. **어떤 화면이 어떤 셸을 입는지** 한눈에. 페이지마다 셸을 따로 감싸지 않고 그룹 레이아웃이 공통 셸을 책임진다.

```
{{/}}                      🔓 공개 셸
├─ {{/login}}              🔓
└─ ({{app}})/              🔒 공통 셸: {{상단바·하단탭 등}}
   ├─ {{/home}}            S1
   ├─ {{/...}}             S{{n}}
   └─ ({{admin}})/         👤 {{관리자}}  · {{미들웨어 게이트}}
      └─ {{/admin/...}}    S{{n}}
```
